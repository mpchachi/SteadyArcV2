# FisherGame + SteadyArc

Este proyecto es una integración de **FisherGame** con **SteadyArc** (Computer Vision). 
El objetivo del juego es pescar mediante diferentes retos físicos capturados a través de la cámara (mediante MediaPipe), sustituyendo los controles clásicos de teclado.

## 🚀 Cómo arrancar el proyecto

Este proyecto está construido con **Next.js** y **React**. Para ejecutarlo localmente en tu ordenador, sigue estos pasos:

### 1. Requisitos previos
Asegúrate de tener instalado [Node.js](https://nodejs.org/) en tu ordenador.

### 2. Instalación de dependencias
Abre una terminal en la carpeta del proyecto y ejecuta:

```bash
npm install
```

### 3. Iniciar el servidor de desarrollo
Una vez instaladas las dependencias, levanta el proyecto ejecutando:

```bash
npm run dev
```

### 4. Abrir el juego
Finalmente, abre tu navegador web y visita:
👉 **[http://localhost:3000](http://localhost:3000)**

---
*Nota: Asegúrate de otorgar permisos de cámara en el navegador si los retos de visión por computadora están activados.*
